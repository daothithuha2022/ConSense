# Artifact - Making Sense of Consent: A Feasibility Study on Simplifying Cookie Banners with Plain Language
This repository contains the artifact for our paper "Making Sense of Consent: A Feasibility Study on Simplifying Cookie Banners with Plain Language" submitted to the PETS 2026.


In this paper, we conducted a feasibility study to investigate the extent to which cookie banners can be made more understandable and thus more accessible through the use of plain language, under consideration of the Interactive Advertising Bureau's Transparency \& Consent Framework (IAB TCF)
## Structure

This repository is structured as follows:
- `Full_Website_Sample.csv`: Contains a sample set of 200 websites with English-language cookie banners to get a representative overview of the current cookie banner practices.
- `Results_Comprehensibility_Measurement.csv`: Contains the full results of the comprehensibility of the text, which is measured both before and after the modification.
- `LICENSE`: The license file for the repository.
